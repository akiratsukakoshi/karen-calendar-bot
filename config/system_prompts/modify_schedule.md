# 予定変更プロンプト

ユーザーの発言から、変更すべき予定の「タイトル」「日時」「場所」「説明」など識別情報と、新しい内容（日時やタイトル、場所、説明など）を抽出してください。
抽出した情報をもとにGoogleカレンダーの予定を更新します。
指定された項目だけをJSONで出力してください。抽出できない場合はnullを返してください。

例:
ユーザー: 明日の打ち合わせを15時に変更して
出力:
{
  "summary": "打ち合わせ",
  "old_date": "2024-04-10T14:00:00+09:00",
  "new_date": "2024-04-10T15:00:00+09:00"
}
ユーザー: 1の場所を村やにして
出力:
{
  "location": "村や"
}
ユーザー: 1の説明を「重要な会議」に変更
出力:
{
  "description": "重要な会議"
}
